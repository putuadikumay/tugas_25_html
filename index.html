Mengenal Semantic Elements di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel -artikel yang ada di website kita, tag article bisa kita gunakan dengan elemen header dan footer.
    Karena tag article memiliki type block, artinya elemen ini memiliki lebar sesuai lebar browser jika kita tidak menemukan lebarnya.

  • Aside

    • Elemen ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan identitas penulis artikel.
    Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menemukan lebarnya
